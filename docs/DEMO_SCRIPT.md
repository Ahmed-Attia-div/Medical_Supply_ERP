## 📦 **المخزون (3 دقائق)**

### الخطوة 1: استعراض
```
"دلوقتي نشوف صفحة المخزون..."

[اضغط على "المخزون" في القائمة]

"هنا كل الأصناف الموجودة:
- مسامير Pedicle Screws
- قضبان Rods
- أقفاص Cages
- صفائح Plates
- أدوات Tools"
```

### الخطوة 2: الفلترة
```
[اضغط على Filter → اختر "Screws"]

"شوفوا كده، لو عايزين نشوف المسامير بس،
نقدر نفلتر بسهولة.

كمان ممكن نفلتر حسب:
- المادة (Titanium, Stainless Steel, PEEK)
- الحالة (طبيعي، منخفض، راكد)"
```

### الخطوة 3: البحث
```
[اكتب في البحث: "5.5mm"]

"والبحث كمان سريع جداً،
لو عايزين نجيب كل المقاسات 5.5mm."
```

### الخطوة 4: التفاصيل
```
"كل صنف عندنا فيه:
- الاسم والـ SKU
- المادة والمقاس
- الكمية المتاحة
- الحد الأدنى
- سعر التكلفة
- سعر البيع
- هامش الربح"
```
